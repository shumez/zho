## 02.01. 你好

> 你好 `nǐ hǎo` (こんにちは)  

> 早上好 (早安) `zǎo shang hǎo` (`zǎo ān`)　(お早うございます)  

> 晚上好 `wǎn shang hǎo` (こんばんは)  

> 晚安 `wǎn ān` (おやすみなさい)  

> 明天见 `míng tiān jiàn` (また明日)  
> 再见 `zài jiàn` (さようなら)

|               |       |               |
|---------------|-------|---------------|
| `nǐ`          | 你    | あなた        　|
| `hāo`         | 好    | 良い          　|
| `zǎo shang`   | 早上  | 朝              |
| `wǎn shang`   | 晚上  | 夜              |
| `wǎn'ān`      | 晚安  | おやすみなさい    |
| `míng tiān`   | 明天  | 明日            |
| `jiàn`        | 见    | 会う          　|
| `zài jiàn`    | 再见  | さようなら       |
| `yī`          | 一    | 1              |
